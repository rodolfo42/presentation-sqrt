Como chegar ao resultado?

  • Para a raiz quadrada de x
  • Começe com uma estimativa y (ex: 1)
  1. Verifique se o resultado (y * y) é satisfatório
    • Se sim, achou
    • Se não, y se torna y / (x / y) e repete

Quê??

Explicando melhor

A média de y e x/y precisa ser igual ou aprox. a y

Exemplo

√ 16
x = 16
(y + 16 / y) / 2 = y
			
y = 4 (rapidin')
(4 + 16 / 4) / 2 = 4
(4 + 4) / 2 = 4
8 / 2 = 4
			
4 = 4 ✔

Implementando

Javascript

function abs(n) {
  return n  > 0 ? n : n * (-1);
}
			
function sqrt(n) {
  var iter = function(n, atual) {
  if( abs((atual * atual) - n) < 0.000000001 )
    return atual;
  else
    return iter(n, (atual + n / atual) / 2);
  }

  return iter(n, 1.0);
}
